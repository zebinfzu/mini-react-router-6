# 实现简单的 React-router-6

## 项目初始化

```bash
yarn create vite mini-react-router-6 && cd mini-react-router-6
yarn install
yarn add react-router-dom@6
yarn dev
```

mini-react-router 目录在 src/mini-react-router

需要在页面上查看每小节演示，修改 main.jsx 中导入的文件即可
